RC Tools へようこそ

RC Tools は wplace の道路敷設・地図反映作業を効率化するためのツール集です。
必要なツールを選んでご利用ください。

道路地図 URL 変換ツール

wplace道路地図のURLと現地のURLを相互に変換できます。

折れ線プランナー

始点と終点の座標から道路の折れ点を計算し、折れ線の作成が容易にできます。

ペイント量計算ツール

2地点のURLから、xy方向の距離と必要なペイント量(px)を計算します。(傾きが1を跨ぐ道路には使用できません)
